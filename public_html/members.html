Members only

After logging into Serif Web Resources with your Email address and Password, select “Dashboard” from the right-hand menu.

To add to the “News” Panel on the front, Home, page simply select “St Josephs Parish News (News)” the FOURTH item in the list. Add any new text on the NEXT line AFTER the . (fullstop) following the joseph dot highgate email address.

NB: Do not overwrite the horizontal rule several line-breaks below as that is your de-limiter for the end of this news panel.

____________________________________

To add to the “Bulletins” Panel on the Bulletins page, you must have FIRST uploaded a PDF (not Word) file of your latest Bulletin to the StJosephs server via FTP.  (Use the FTP links as per the separate email you will have been sent).

From your FTP interface, once you have successfully logged-in, double-click on the “stjosephshighgate.org.uk” folder to open it. Then double-click on the “public_html” folder to open it.
Then scroll DOWN the list of files you see. Do NOT click-on, or remove or change ANY of these files.
At the bottom of the list you will see a Folder called “Bulletins” double-click on that folder to open it. You will see the existing PDF files listed in numbered order. It is important that you have named the NEW file you are uploading in EXACTLY the same way and that you don’t forget to add the “.pdf” extension to the end of your file name.

Then drag-and-drop your new file FROM your hard-disk over into this folder.

Once copied, simply exit your FTP uploader and return to the “Bulletins” panel on your Serif Web Resources Dashboard. (The THIRD item in the list.)

Add a NEW line (ie “Bulletin 15/11/2015”) then select the “Insert/edit link” (the one just to the left of the icon with the “mountain” outline in a box) and paste the following link into the “Url” space: “ http://stjosephshighgate.org.uk/Bulletins/bulletin_xxxxxxxx.pdf “
NB: the xxxxxxxx = date month year (in full), ie 15112015
Then scroll down and select “Save” or select “Save and Exit”
if you are completely finished and wish to exit Serif Web Resources completely.

Then “Log Out” from Serif Web Resources if the “Save and Exit” selection has not already done so.

NB: See vital information to below***!!

***It is possible to insert picture files into the Latest News and Bulletin updates, HOWEVER, it is essential that all images must first have been re-sized and scaled in an Image Editing program (such as Photoshop for example) to no wider than 300 pixels.

Should this rule not be followed, it may cause the panels to freeze and corrupt and crash the website’s html.

So proceed with caution. If you are not sure, please email the administrator before attempting to upload or change.
